# Royal Catalogue Management

## Installation
```
git clone git@github.com:kojinkai/electron-react-starter.git your-folder-name --depth=0
```
Then
```
yarn install
```
Then finally run
```
yarn start
```

This will start the Webpack dev server with everything from create-react-app baked in.
